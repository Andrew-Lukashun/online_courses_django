# Online Courses Platform (Django)

Веб-платформа для управления образовательным контентом с многоуровневой иерархией. Проект реализует полный цикл управления курсами: от структурирования программы до хранения учебных материалов.

## Ключевой функционал
* **Иерархическая декомпозиция**: Организация материалов по принципу `Курс` -> `Часть` -> `Тема`.
* **Управление медиа-активами**: Система загрузки и хранения учебных документов и файлов с привязкой к конкретным модулям.
* **Централизованное администрирование**: Использование кастомизированной Django Admin Panel для управления контентом и пользователями.
* **Data Integrity**: Использование Django ORM для обеспечения целостности данных и управления связями в БД.

## Технологический стек
* **Framework**: Django 4.x / 5.x
* **Language**: Python 3.10+
* **Database**: PostgreSQL (или SQLite для разработки)
* **Frontend**: Django Templates, HTML5, CSS3

## Архитектура проекта
Проект разделен на логические модули:
* `courses/` — основное приложение для управления курсами и контентом.
* `media/` — хранилище загружаемых документов и файлов курсов.
* `templates/` — HTML-шаблоны для отображения интерфейса.
* `core/` (или `config/`) — настройки проекта и WSGI/ASGI конфигурации.

### Установка и запуск

### 1. Подготовка окружения
Клонируйте репозиторий, перейдите в нужную ветку и создайте виртуальное окружение:
```
git clone [https://github.com/Andrew-Lukashun/online_courses_django.git]
cd online_courses_django
git checkout django
python -m venv venv
```

Активируйте виртуальное окружение и установите зависимости:

Для Windows:
```
venv\Scripts\activate
```
Для Linux/macOS:
```
source venv/bin/activate
```
```
pip install -r requirements.txt
```
### 2. Настройка базы данных и миграции
Выполните миграции, чтобы создать структуру таблиц в базе данных:
```
python manage.py migrate
```
Создайте суперпользователя для доступа к админ-панели:
```
python manage.py createsuperuser
```
### 3. Запуск приложения
Запустите сервер разработки Django:
```
python manage.py runserver
```
